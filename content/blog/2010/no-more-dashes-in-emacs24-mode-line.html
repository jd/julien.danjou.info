--- title: No more dashes in Emacs 24 mode-line created: 2010-10-20 13:01:00 image: /media/images/blog/2010/emacs-no-dashes.png tags: - Emacs --- {% block excerpt %} {% mark excerpt %} We all know the good old Emacs mode-line you got under every window. Since the beginning (a long time ago), it starts and ends with dashes. I've proposed [a patch](http://lists.gnu.org/archive/html/emacs-devel/2010-10/msg00675.html) to remove them. {% endmark %} {% endblock %} Before: ![Screenshot of Emacs with dashes in the mode line]({{site.url}}/media/images/blog/2010/emacs-dashes.png) After: ![Screenshot of Emacs without dashes in the mode line]({{site.url}}/media/images/blog/2010/emacs-no-dashes.png) This has been merged in Emacs 24. You won't see any more ugly dashes in graphical mode.
