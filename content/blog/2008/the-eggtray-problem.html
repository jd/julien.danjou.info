--- title: The eggtray problem created: !!timestamp '2008-10-03 00:00:00' tags: - x11 - gtk - systray - freedesktop --- {% mark excerpt %} I still don't know why but many GTK+ applications use something called eggtrayicon. As far as I know, *eggtrayicon.c* is a file written in 2002 by Anders Carlsson which implements the [Freedesktop.org system tray](http://standards.freedesktop.org/systemtray-spec/latest/) procotol for GTK+ applications. {% endmark %} Problem is that this C file is used in dozens of programs and maybe more, and is a bit bugged. I've already send patches for [mail-notification](http://www.nongnu.org/mailnotify/) and [Audacious](http://audacious-media-player.org). [pidgin](http://www.pidgin.im/) is the first fixed implementation I found and works quite well. Many other applications are probably affected. That seems to me like a real problem.
